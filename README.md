# M. Caner Arslan Portfolio

A single-page portfolio website for M. Caner Arslan that highlights experience, skills, and projects. The layout uses semantic HTML and modern CSS to provide a responsive, professional feel.

## Getting started
1. Open `index.html` in any modern browser to view the live page.
2. Modify `style.css` to adapt colors, typography, or spacing if needed.

## Stack
- HTML5
- CSS3
- Remix Icon (for the icons in the header and footer)

## Notes
- All assets are located inside the `assets` directory (e.g., favicon).
- This project is intentionally simple and can be served with any static host.
